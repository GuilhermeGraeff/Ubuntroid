# Ubuntroid
Minha tentativa de fazer um dualboot num samsung s21fe todo quebrado, já que n sei como, vou guardando aqui o que consigo

A ideia é trazer para mais perto do usuário convencional a experiência de estar imerso em um sistema linux, com o intuito de democratzar o conhecimento sobre tecnologia voltada para a infraestrutura, desenvolvimento e teoria daquilo que utilizamos.

A ideia e manter os dois sistemas separados para não haver a troca de maneira dinâmica, como propósito, se quiser trocar de sistema operacional, reinicie o sistema.

Imagina utilizar computadores portáteis sem a inetrface interface gráfica, apenas comandos, acredito que isto seja um passo para o futuro, o que eu deslumbro.


(compreendo a controvérsia em unir os nomes, de um lado Ubuntu(unix) do outro Android (unix), porém o nome é para identificar ambos e não por acaso o Ubunt vem primeiro, e também nomeia o projeto/ideia geral do que eu penso que é ciência.)
